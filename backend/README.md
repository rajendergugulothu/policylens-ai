# PolicyLens AI — Backend API

FastAPI backend for PolicyLens AI. Pre-production compliance testing for AI agents.

## Environment variables (set in Space settings)

| Variable | Description |
|----------|-------------|
| `DATABASE_URL` | PostgreSQL connection string (get from Neon or Supabase) |
| `ANTHROPIC_API_KEY` | Your Anthropic API key |

## API docs

Once deployed, visit `/docs` for the interactive Swagger UI.
